# Prompts pour Nano Banana Pro (Gemini Image Generation)

Pour obtenir le meilleur résultat "Avant / Après" avec la même structure de pièce, il est crucial de garder la description de l'architecture **identique** dans les deux prompts.

Si possible, essayez de fixer une "seed" (graine aléatoire) si l'outil le permet, pour geler la structure de l'image.

## 1. Prompt "Avant" (Pièce Vide)

> **Copiez ce prompt :**
>
> "Photorealistic 8k image of a spacious empty living room in a modern Parisian apartment. White walls with intricate moldings, light oak chevron parquet flooring. Large floor-to-ceiling french windows on the left letting in bright natural sunlight. No furniture, empty space, clean architecture, wide angle shot, neutral lighting, high resolution, architectural photography."

## 2. Prompt "Après" (Pièce Meublée)

> **Copiez ce prompt :**
>
> "Photorealistic 8k image of a spacious **furnished** living room in a modern Parisian apartment. **Furnished with a contemporary beige linen sofa, a glass coffee table, a large soft area rug, and lush green potted plants.** White walls with intricate moldings, light oak chevron parquet flooring. Large floor-to-ceiling french windows on the left letting in bright natural sunlight. **Warm and cozy atmosphere, interior design magazine style**, wide angle shot, high resolution."

---

### Conseils pour la cohérence :
1.  **Ne changez pas** les mots clés structurels (windows, floor, walls, angle).
2.  L'ordre des mots est important, gardez le début similaire si possible.
